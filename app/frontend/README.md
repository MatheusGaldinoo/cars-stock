# Frontend - Leal Car 🚗

Este é o frontend do sistema de gerenciamento de estoque de carros, desenvolvido com **React**, **Vite**, **TypeScript** e **Foundation CSS**.

## Pré-requisitos
- [Node.js](https://nodejs.org/) (versão 18+ recomendada)
- O **backend** (`app/backend`) deve estar rodando para que a API funcione perfeitamente.

## Como Executar

### 1. Instale as dependências
Abra o terminal na pasta `app/frontend` e execute:
```bash
npm install
```

### 2. Inicie o servidor de desenvolvimento
Ainda na pasta `app/frontend`, rode:
```bash
npm run dev
```

### 3. Acesse a Aplicação
Abra o seu navegador e acesse a URL fornecida no terminal (geralmente [http://localhost:5173](http://localhost:5173)).

---

## Funcionalidades
- **Visualização de Estoque:** Mostra todos os carros disponíveis e vendidos, utilizando componentes de cartões (post-cards).
- **Barra de Pesquisa:** Permite localizar um veículo específico informando a Placa.

## Tecnologias Utilizadas
- **React 18**
- **Vite** (Build Tool super rápido)
- **TypeScript**
- **Axios** (Integração com a API Backend)
- **Zurb Foundation Sites** (Framework CSS) + **SASS / SCSS**
